## Before the Interview

Run `progress-checklist.md` the evening before. Time a 90-second explanation of M01 without notes. If that fails, re-read M01 and its Open Book links.

面試前一晚執行 `progress-checklist.md`。在不看筆記的情況下，以 90 秒說明 M01。若無法完成，重讀 M01 及其 Open Book 連結。
